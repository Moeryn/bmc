BMC – Braille Music Compiler

Disclaimer

This is an experimental project. As such, it is not ready for end users. Many of the technologies in use currently are not explored fully yet and might still be identified as inappropriate for the task at hand. In other words: This project is under construction! However, if you are interested in programming and braille music, this disclaimer should not discourage you from contributing.

Why C++?

Because C++ is a great language, and allows for compilation to native code on basically every platform in existance.

One big disadvantage of using C++ to implement a cross-platform accessible program is that there is no cross-platform GUI library, a deficiency that Java does solve nicely.

Goals

  • Parse braille music code:
    • Resolve value ambiguities
    • Expand braille repeation instructions as well as simile signs
    • Calculate octaves for all notes and intervals
  • Output:
    • To MusicXML
    • To MIDI

Boost libraries

All boost dependencies are header only. This means that it is not necessary to compile your version of boost, you can simply point the build system to the toplevel directory containing the boost distribution.

The following Boost libraries are used in this project.

  • Fusion (BOOST_FUSION_ADAPT_STRUCT)
  • Range (boost::range::insert)
  • Rational
  • Spirit
  • Test

Braille tables

The braille table (ttb) parsing code has been taken from BRLTTY and stripped down a little to remove unused functionality.
